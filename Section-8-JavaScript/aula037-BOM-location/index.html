Método location.href

Força o rediresionamento da página;

Método location.host

A propriedade location.host retorna o host (endereço IP ou domínio) e a porta de uma URL.

Método location.hostname

A location.hostnamepropriedade retorna o host (endereço IP ou domínio) de uma URL. A location.hostname propriedade também pode ser definida para navegar para o mesmo URL com um novo nome de host.

Método location.origin

A origin propriedade retorna o protocolo, o nome do host e o número da porta de uma URL. A origin propriedade é somente leitura.

Método location.pathname

A pathname propriedade define ou retorna o caminho de uma URL (página).

Método location.port

A port propriedade define ou retorna o número da porta de um URL.

Note:Se o número da porta for padrão (80 para http e 433 para https), a maioria dos navegadores não retornará nada.

Método protocol

A propriedade protocol define ou retorna o protocolo da URL atual, incluindo os dois pontos (:). O protocolo é um padrão que especifica como os dados são transmitidos entre computadores.

Método location.search

A search propriedade retorna a parte da string de consulta de uma URL, incluindo o ponto de interrogação (?). A search propriedade também pode ser usada para definir a string de consulta.

Método location.assign

O assign() método carrega um novo documento.

Note A diferença entre assign() e replace(): replace() remove o URL atual do histórico do documento. Com replace() não é possível usar "back" para navegar de volta ao documento original.

Método location.replace

O replace() método substitui o documento atual por um novo.

Método location.reload

O reload() método recarrega o documento atual. O reload() método faz o mesmo que o botão de recarregar do seu navegador.
