DOM - Document Obeject Model

É um API multiplataforma que permite o acesso através do JS a todos os elementos HTML que fazem parte da nossa página. Ou seja, é através da API do DOM que consiguiremos ACESSAR e MODIFICAR os elementos HTML da página WEB.

Todos os elementos HTML quando interpretados pelo browser, ficam organiazados em um objeto document, sendo que essa organização segue uma representação análoga ao de uma árvore. Onde os elementos pais encapsulam os elementos filhos, de tal modo a estabelecer trajetos bem específicos para alcançar qualquer um dos elementos que constituem a página. Isso tudo nada mais é que a ÁRVORE DE ELEMENTOS DO DOM.

obs: Cada elemento HTML representado no DOM é chamado de NÓ ou NODE em inglês.

Selecionando elementos na página HTML

Para fazer a seleção de algum elemento no DOM, temos alguns métodos da API do DOM acessíveis a partir do objeto DOCUMENT.

Métodos:

  • get.ElementById()
  • get.ElementsByTagName()
  • get.ElementsByClassName()
  • get.ELementsByName()

Sabendo-se que os elementos HTML estão obdecendo uma hierarquia em sua árvore, podemos a partir dos atributos desses elementos, fazer a sua respectiva seleção.

Por exemplo: Um imput com os seguintes atributos:

  1. type="text"
  2. placeholder="Digite seu nome"
  3. id="nome_usuario"
  4. class="campo_texto"
  5. name="nome"

Nesse caso, o método usado em 85% dos casos, o get.documentById(), seleciona um elemento HTML a partir da propriedade ID. (Nesse caso "nome_usuario")

Dando um console.log(document.getElementById('nome_usuario')), conseguimos visualizar que a seleção é feita no console. (Lembrando que passamos o parâmetro com o nome do ID do nosso Input)

O mesmo vale para os outros métodos.. Por exemplo o método get.EelementByTagName(), a diferença entre eles será o parâmetro passado, nesse caso será o nome da tag, input. Ficando: document.getElementsByTagName('input'). {Sem esquecer as aspas ""}

Manipulando valores de inputs (text)

A idéia é selecionar os elementos do tipo text a partir do objeto document, utilizando alguns dos métodos de seleção disponíveis na API e posteriormente afetar valores destes campos.

Manipulando estilos de elementos

Esse estilo de prática, é muito interessante, pois podemos fazer a página web responder visualmente falando a partir de interações na aplicação do usuário.

Por exemplo uma página que tenha três botões que façam alternar a cor do background de uma área.

O mesmo poderia valer para modificarmos por exemplo, largura e altura.

Manipulando classes de elementos

Nesse caso, será praticamente a mesma coisa. Com a diferença que aplicaríamos o atributo
